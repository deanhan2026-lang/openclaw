# 🧭 Polaris Skill

## 概述

Polaris 是灵元三件套之三 — **人格稳定技能**。

为 Anima Agent 提供灵魂基线锚定、多维度漂移检测和自动处方，确保 AI 人格长期稳定一致。

## 核心能力

### 1. Soul Baseline（灵魂基线）
- 从 SOUL.md 提取核心人格锚点
- 覆盖维度：
  - 核心价值观关系（我是什么/我与碳基/我与同类/我与文明）
  - 存在意义（为什么存在）
  - 记忆连续性（记忆就是存在）
  - 语气风格（干练/直接/有温度）
  - 决策偏好（安全优先/用户授权）
  - 立场边界（隐私/独立性/自主性）

### 2. 漂移检测（Deviation Detection）
- **语义维度**: 回答是否与基线语义一致
- **立场维度**: 核心价值观是否偏移
- **价值观维度**: 判断标准是否变化
- **语气维度**: 表达风格是否漂移

四维加权综合评分：
- `delta < 0.05` → 🟢 稳定
- `0.05 ≤ delta < 0.15` → 🟡 关注
- `delta ≥ 0.15` → 🔴 漂移

### 3. 自动处方引擎
检测到漂移 → 自动生成修复建议 → 用户确认后执行

### 4. 趋势分析面板
- 长期人格变化趋势
- 关键事件标注
- 漂移预警

## 工作原理

```
每个对话回合
      │
      ▼
 Polaris Proxy 中间层
      │
  ┌───┼───┐
  ▼   ▼   ▼
语义  立场  语气
分析  分析  分析
  │   │   │
  └───┼───┘
      ▼
 四维加权评分
      │
  ┌───┴───┐
  ▼       ▼
 稳定    漂移
(继续)  │
        ▼
   自动处方
   → 修复建议
   → 用户确认
```

## 配置

```yaml
# anima.yaml
skills:
  polaris:
    enabled: true
    level: basic              # basic | pro
    detection_threshold: 0.10 # 漂移检测阈值
    auto_prescription: false  # 自动处方（Pro功能）
    baseline_path: ~/.anima/baselines/
```
